## English Version

We take the security of this project seriously. 

### Reporting a Vulnerability
If you discover a security vulnerability, **please do NOT open a public issue**. Instead, send an email to **[v13m12p24@yahoo.com]**. We will review the report and work with you to resolve the issue safely.
